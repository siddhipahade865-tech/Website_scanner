# Website Vulnerability Scanner

## About

Website Vulnerability Scanner is a simple Python-based cybersecurity project that performs basic security checks on a website. It analyzes the website and provides a report about common security features and possible weaknesses.

## How It Works

The user enters a website URL, and the program sends a request to the website. 
It then checks for HTTPS support, server information, and common HTTP security headers before displaying the results.

## Features

* Checks if the website is reachable
* Detects HTTPS usage
* Analyzes common security headers
* Displays basic server information
* Generates a simple security report

## Technologies Used

* Python 3
* Requests library
* Basic Networking Concepts

## Purpose

The purpose of this project is to practice Python programming and understand the basics of web security and networking.
